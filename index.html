Home / Quick Start Guide

Quick Start Guide

Get up and running with DNSRedo in minutes

5-Minute Setup: Follow these steps to secure your first domain with DNSRedo protection.

Step 1: Login & Authentication

 Authentication Method: Google OAuth
 Access: Web interface, mobile-friendly
 Setup Time: < 30 seconds

Click "Login with Google" to authenticate using your Google account. No password management required.

Step 2: Add Your First Domain

Supported Providers

  • Cloudflare (recommended)
  • AWS Route53
  • DigitalOcean
  • Namecheap

Required Information

  • Domain name
  • DNS provider
  • API credentials
  • Account permissions

Adding a Domain:

  1. Navigate to Domains tab
  2. Click "Add Domain" button
  3. Select your DNS provider
  4. Enter domain name and API credentials
  5. Test connection and save

Step 3: Enable Protection

Important: Enable protection immediately to start monitoring DNS changes and creating automated backups. 
 Location: Security Center → Monitoring Tab
 Auto-backup: Every 5-60 minutes
 Real-time alerts: Instant notifications
 Email notifications: Configurable

Step 4: Create Initial Backup

Create your first manual backup to establish a baseline:

  1. Go to Security Center → Backups
  2. Click "Create Backup"
  3. Add description (e.g., "Initial baseline backup")
  4. Verify backup completion

Step 5: Configure Alerts

Alert Types

  • DNS record changes
  • Propagation failures
  • Security threats
  • Performance issues

Notification Methods

  • Email notifications
  • In-app alerts
  • Dashboard warnings
  • API webhooks (coming soon)
You're All Set! Your domain is now protected with automated monitoring, backups, and intelligent alerts. Visit the Analytics page to monitor performance.
